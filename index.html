Toxicity Prediction using Molecular Transformers

Built an advanced toxicity prediction pipeline that leverages Transformer-based models for analyzing chemical compounds represented in SMILES (Simplified Molecular Input Line Entry System) format. By framing the problem as a sequence classification task, the project utilizes pretrained Transformer architectures to learn deep molecular representations, enabling accurate binary toxicity classification.

Challenges & Solutions: One major challenge was handling complex chemical representations in SMILES format, which required preprocessing and tokenization strategies tailored for molecular strings. Traditional ML approaches struggled to capture the sequential patterns, so we integrated Hugging Face Transformers for a contextual understanding of the molecules. Another difficulty was the limited amount of labeled data, which was mitigated through data augmentation techniques and transfer learning using pretrained Transformer models. Overfitting and long training times were addressed using dropout, gradient clipping, and learning rate scheduling. Additionally, extensive hyperparameter tuning and model evaluation were performed to ensure stability and generalization.
